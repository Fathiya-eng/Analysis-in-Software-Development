5.2 User Stories with Acceptance Criteria
US01 – Create Bank Account

As a new customer
I want to create an online bank account
So that I can access banking services

Acceptance Criteria:

User must provide valid national ID

Email/phone verification required

Account number generated automatically

Story Points: 5
Priority: High

US02 – Customer Login

As a customer
I want to log in securely
So that I can access my account

Acceptance Criteria:

Login requires email/phone + password

Incorrect login attempts lock the account

Supports two-factor authentication

Story Points: 8
Priority: High

US03 – View Account Balance

As a customer
I want to view my balance
So that I can know my available funds

Acceptance Criteria:

Balance updates in real time

Shows available and total balance

Story Points: 3
Priority: High

US04 – Transfer Money

As a customer
I want to transfer money
So that I can send payments

Acceptance Criteria:

Transfer cannot exceed available balance

User must confirm the transfer

Transaction reference number generated

Story Points: 8
Priority: High
